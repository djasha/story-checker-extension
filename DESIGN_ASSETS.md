# Story Checker Branding & Assets

This document outlines the visual identity and provides specifications for the Chrome Web Store media assets.

## Visual Identity

| Element | Specification |
|---------|---------------|
| **Primary Color** | `#4f46e5` (Indigo) - Main UI and Backgrounds |
| **Accent Color** | `#10b981` (Emerald) - Success states & "YES" icons |
| **Warning Color** | `#f59e0b` (Amber) - Special badges |
| **Typography** | Sans-serif (Inter/Roboto/Segoe UI) |

## Logo Concept
The logo features a **Superellipse** (rounded square) background in an Indigo gradient. Inside is a white circular **Story Ring** containing a thick white **Checkmark**.

## Required Store Assets
